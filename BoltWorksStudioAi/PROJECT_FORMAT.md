# BoltWorks project format

## Canonical wrapper

```json
{
  "kind": "modeler-project",
  "version": 1,
  "name": "asset-name",
  "savedAt": "2026-07-21T12:00:00.000Z",
  "textureLibrary": [],
  "scene": {
    "version": 2,
    "coordinateSystem": {},
    "shapeConventions": {},
    "groups": [],
    "objects": [],
    "boneHierarchy": []
  },
  "editor": {}
}
```

The loader also accepts a legacy scene object with a top-level `objects` array,
but AI-generated deliverables should use the wrapper because it preserves
textures and editor state.

## Coordinate contract

- right-handed coordinate system;
- `+Y` is up;
- ground plane is XZ, normally at `Y = 0`;
- world position arrays are `[x, y, z]`;
- Euler rotation arrays are `[x, y, z]` in degrees, order XYZ;
- scale arrays are local `[x, y, z]` multipliers.

Current saved scenes declare `coordinateSystem.units` as `"meters"`. Treat the
numeric values as editor scene units and preserve them exactly while editing a
project. Do not independently apply the Roblox/DAE conversion. Exporters own
that conversion; DAE uses the studio rule `1 stud = 0.28 meters`.

## Groups

```json
{
  "id": "roof",
  "name": "Terracotta Roof",
  "parentId": "cottage"
}
```

- Group IDs must be unique.
- `parentId` is null for a root group or references another group ID.
- Cycles are invalid.
- An object's `groupId` references a group.
- `groupName` is a readable fallback but does not replace a valid `groupId`.

## Objects

Minimum recommended object:

```json
{
  "id": "wall-front",
  "name": "Front wall",
  "shape": "box",
  "position": [0, 1.5, 2],
  "rotation": [0, 0, 0],
  "scale": [4, 3, 0.2],
  "color": "#c7bda5",
  "roughness": 0.85,
  "opacity": 1,
  "hidden": false,
  "groupId": "walls",
  "groupName": "Walls"
}
```

Supported object fields:

| Field | Meaning |
|---|---|
| `id` | Stable unique identifier. Required for reliable selection and editor state. |
| `name` | Human-readable component name. |
| `shape` | Built-in shape key. See `SHAPE_CATALOG.md`. |
| `geometry` | Optional custom BufferGeometry data: `positions`, optional `normals`, `uvs`, `indices`. |
| `position` | World-space XYZ numbers. |
| `rotation` | XYZ Euler degrees. |
| `scale` | Local XYZ scale. Keep all components positive unless a deliberate mirrored mesh was tested. |
| `color` | Hex material color such as `#40c7a5`. |
| `roughness` | Usually `0` to `1`. Low is glossy, high is matte. |
| `opacity` | `0.05` to `1`. Values below `1` enable transparent rendering and disable depth writing; use about `0.38` for leaded window glass. |
| `hidden` | Saved visibility flag. |
| `groupId`, `groupName` | Scene hierarchy membership. |
| `pivot` | Optional object-local XYZ pivot. |
| `linkId`, `linkColor` | Optional editing/link metadata. |
| `cuts` | Optional stored cut description generated by the studio. |
| `bevel`, `depth`, `direction` | Parameters used by supported generated geometry. |
| `textureName` | Name in project `textureLibrary`. |
| `textureUrl` | Runtime image URL. Use null when textureName points to embedded data. |
| `textureFlipY` | Boolean texture orientation. |
| `textureRotation` | Degrees normalized to 0–359. |
| `textureRobloxAssetId` | Optional Roblox asset reference. |
| `materialRule` | Material/export rule; use `"auto"` unless the workflow specifies another value. |

If `geometry` exists, it takes precedence over the built-in shape geometry.
`positions` contains flat XYZ triples. `uvs` contains flat UV pairs. Indices are
optional. The studio computes normals when valid normals are not supplied.

## Texture library

```json
{
  "name": "Mossy Fieldstone",
  "dataUrl": "data:image/png;base64,iVBORw0KGgo...",
  "robloxAssetId": ""
}
```

Names are unique lookup keys. A project is portable only when required images
are embedded as data URLs. Keep one library entry per distinct bitmap.

## Editor state

`editor` is optional to the geometry but recommended for a polished project.
Important fields are:

```json
{
  "projectName": "asset-name",
  "selectedId": null,
  "selectedGroupId": null,
  "checkedIds": [],
  "activeGroupIds": [],
  "activeTransformMode": null,
  "facePickMode": false,
  "view": {
    "cameraPosition": [8, 6, 8],
    "orbitTarget": [0, 1.5, 0],
    "cameraUp": [0, 1, 0],
    "viewSpace": 1.5,
    "shotZoom": 0.85,
    "environment": "plain",
    "background": "plain",
    "showGrid": true,
    "useCurrentZoomInShots": false,
    "hideGridInShots": true
  }
}
```

Valid environments are `plain`, `studio`, `road`. Valid backgrounds are
`plain`, `studio`, `sky`, `sunset`.
